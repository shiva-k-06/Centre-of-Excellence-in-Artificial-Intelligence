High-Performance Computing System

NVIDIA DGX-1 Server
  • GPU: 8 × NVIDIA Tesla V100 — 32 GB HBM2 memory per GPU (Total: 256 GB)
  • CPU: Dual Intel Xeon Processors
  • System Memory: 512 GB
  • Interconnect: NVLink GPU interconnect
  • Storage: High-speed SSD storage array
  • OS: DGX OS Server (Ubuntu-based)
  • Purpose: Large-scale AI model training, simulation, and research computing

Workstation Cluster

6 High-Performance Workstations
  • Count: 6 workstations
  • Processor: Intel Core i7-9700
  • RAM: 16 GB DDR4 (3200 MHz)
  • Storage: 256 GB SSD + 2 TB HDD
  • GPU: NVIDIA RTX 2080 Ti (11 GB GDDR6)
  • OS: Windows 10 Pro (64-bit)
  • Purpose: Data preprocessing, model development, visualization, and inference

Power Backup System

20 kVA Industrial Online UPS
  • UPS: 20 kVA Industrial Online UPS
  • Battery: Amaron Quanta 12V 75Ah SMF-VRLA batteries
  • Units: 14 battery units available
  • Backup: Approximately 2–3 hours under full load
  • Purpose: Uninterrupted power supply for DGX-1 server and workstation cluster
